To run the bison and flex commands we have written a makefile.So to create a executable file run the following command 
make
Now an executable named assn4 is created.Run it through the command  
./assn4
Now give the required input(s)
After giving the inputs press Ctrl+d to make terminal know we have completed giving the input.Syntactical correctness is displayed on the terminal right after giving the input.
Now a cpp file named ans.cpp is created which when compiled and executed prints the result of queries given to do so run the following:
g++ ans.cpp
./a.out
To remove all the new files created and start over run the following:
make clean
